Web数据挖掘

Apr 11, 2017 | Web | Hits

Web数据挖掘是在数据挖掘技术的基础上，针对网络数据，主要是Web文档和服务日志文件等数据进行分析、归纳和汇总，并在其中发现和提取潜在的有用信息及知识的技术。

Web数据挖掘概述

Web内容挖掘

　　Web内容挖掘指从Web网站的内容中发现潜在的有价值的信息和抽取知识的过程。Web网站的内容分为：文本、图像、音频、视频和动画等。因此，Web 内容挖掘又可分为文本数据挖掘（包括文本和网页文件格式）和多媒体数据挖掘（图像、音频、视频等多媒体数据），Web内容挖掘主要应用于文献检索系统、垃圾邮件的处理、信息过滤、电子资源检索等。（此处以文本挖掘为例）

Web结构挖掘

　　Web 结构挖掘是对网站的超级链接进行分析和处理，从各个超级链接中发现权威网页和有价值的网页，并根据此标准对网页和网站进行评级、分类。Web结构挖掘主要任务是查找权威网页，并识别各网页之间的相互关系，从而发现网站的结构，获取不同网页相似度和网站的关联度。Web结构挖掘主要应用于搜素引擎，对同一主题的网页进行排序。接下来介绍两种著名的网页排名算法。

PageRank算法

　　PageRank算法是一种忽略掉Web页面上的文本和其它内容，根据网页之间相互的超链接来计算网页排名的一种技术。Google用它来体现网页的相关性和重要性，在搜索引擎优化操作中是经常被用来评估网页优化的成效因素之一。　　
　　PageRank的核心思想：
　　1）如果一个网页被很多其他网页链接到的话说明这个网页比较重要，也就是PageRank值会相对较高；
　　2）如果一个PageRank值很高的网页链接到一个其他的网页，那么被链接到的网页的PageRank值会相应地提高。

HITS算法

　　HITS算法是一种根据用户的查询请求来分析网页重要性的算法，此算法认为每一个网页应该将其authority权重和hub权重分开来考虑。
　　HITS核心思想：
　　１）一个高质量的hub页面会指向很多高质量的authority页面。
　　２）一个高质量的authority页面会被很多高质量的hub页面所指向；

Web使用挖掘

　　Web 使用（日志）挖掘是通过对计算机上的日志文件，如在服务器的Log 文件和cookies 文件等，涉及关联规则的挖掘，从中发现频繁使用的网页模式。Web日志挖掘的主要目的包括网络广告分析、流量、用户分类、网络欺骗预防等等。Web使用挖掘主要应用于商业营销，并为商业决策提供依据。

Web数据挖掘的应用及发展方向

　　在搜索引擎中的应用：利用Web挖掘技术，可以提高搜索引擎获取信息与反馈结果的质量，更好地满足用户的个性化需求。例如：

  • 页面的自动分类

  • 权威页面的发现

  • 用户兴趣偏好的挖掘

　　在电子商务中的应用：挖掘电子商务系统中Web日志与访问内容，可以获得用户的访问模式以及有价值的信息。例如：

  • 用户的兴趣爱好

  • 购买频率

  • 所属用户群及其特征

  • 页面访问情况

  • 广告点击情况

　　Web挖掘的发展方向：

  • 建立新的和更好的分析模型，用于处理数据类型复杂，数据结构比较特殊的数据；
  • 融合各种异构数据的挖掘技术，加强对各种非结构化数据的开采，如图形数据、视频图像数据以及音频等综合多媒体数据；
  • 基于关键词的搜索引擎不能完全理解用户输入的意义，所以给出的页面可能不是用户所需要的。要在这方面取得进展，依赖于自然语言处理的研究进展。
