一、爬虫简介

​ 说到爬虫，那么何为爬虫？爬虫其实就是一段自动抓取互联网信息的程序，爬虫的价值所在就是让互联网数据为我所用。一般而言，爬虫的架构主要分三部分URL管理器、网页下载器、网页解析器，我们先来看一个通用的网络爬虫框架：

​ URL管理器：管理待抓取URL和已抓取URL，防止重复、循环抓取；

​ 网页下载器：将互联网上URL对应的网页内容下载到本地；

​ 网页解析器：从已下载的网页中提取出有价值的数据。

​ 这里有一个疑问？爬虫究竟违不违法？事实是网络爬虫目前还处于早期的“蛮荒阶段”，法律并未明确规定相关细节。从目前的实践来看，如果抓取数据的行为用于个人使用，则不存在问题；而如果将数据用于商业，那么抓取的数据类型就非常关键了，可能会造成侵权，继而违反法律。为此我们在抓取数据时，请限制你的爬虫，并遵守相关网站的要求和协议，大数据虽然便宜，但律师费可不便宜。

二、环境准备

  • Python基础运行环境：安装Python3.X，下载参见官方地址，并安装相关的第三方包。如果怕麻烦，可以安装Anaconda，这是一个用于科学计算的Python发行版，提供了包管理与环境管理的功能，可以很方便地解决多版本Python并存、切换以及各种第三方包安装问题。Anaconda的下载页参见官网下载，Linux、Mac、Windows均支持。
  • Requests：它是基于urllib，urllib2升级版的开源HTTP，使用更加方便，用于下载网页（查看官方文档）。
  • Beautifulsoup：是一个可以从HTML或XML文件中提取数据的Python库，用于解析网页（查看官方文档），此库适用于新手，它可以提取一般简单的数据，但对于复杂的数据则无能为力，这时就需要正则表达式出马了。
  • 集成开发环境：至于IDE的选择，这里推荐两个，Pycharm和Eclipse+PyDev，都挺不错的，Pycharm比较好用，这里推荐使用。此外Python自带编辑器，Anaconda也有Jupyter Notebook和Spyder等，可以自由选择。
